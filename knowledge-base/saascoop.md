# SaaSCoop

## Princípios

- soberania tecnológica
- software livre
- inteligência artificial sustentável
- cooperação
- governança comunitária
- trabalho digno
- fundo perpétuo
- independência de investidores externos

## Critérios técnicos

Uma solução SaaSCoop deve ser:

- aberta quando possível
- documentada
- reutilizável
- barata de operar
- segura
- sustentável
- útil para comunidade
- adequada para formação educacional
