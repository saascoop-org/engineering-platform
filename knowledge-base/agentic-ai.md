# Agentic AI

## Conceitos essenciais

- agente
- ferramenta
- plano
- memória
- contexto
- orquestração
- reflexão
- avaliação
- guardrails
- human-in-the-loop

## Boas práticas

- limitar autonomia por risco
- registrar decisões do agente
- versionar prompts
- definir contratos de entrada e saída
- usar structured outputs quando possível
- criar avaliação automática
- manter fallback humano para decisões críticas

## Anti-patterns

- agente genérico demais
- ausência de avaliação
- ausência de logs
- prompt sem versionamento
- ferramenta sem contrato
- uso de memória sem governança
