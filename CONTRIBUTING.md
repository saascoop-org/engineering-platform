# Contributing

Obrigado por considerar contribuir.

## Como contribuir

1. Abra uma issue descrevendo a melhoria.
2. Discuta a proposta antes de mudanças grandes.
3. Crie uma branch curta e descritiva.
4. Use Conventional Commits.
5. Abra um pull request com contexto, impacto e testes realizados.

## Padrão de qualidade

Toda contribuição deve considerar:

- clareza
- reutilização
- segurança
- governança
- custo-benefício
- documentação
- rastreabilidade
- manutenibilidade

## Conventional Commits

Exemplos:

```text
feat(prompts): add security architect mentor
docs(adr): document project-based architecture
fix(knowledge-base): improve agentops definitions
```
