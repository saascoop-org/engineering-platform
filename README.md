# AI Engineering Toolkit

Família de tutores especializados para orientar a criação de soluções modernas de Inteligência Artificial Agêntica com excelência em arquitetura, engenharia, dados, software, IA, governança, segurança, observabilidade e custo-benefício.

## Objetivo

Este repositório organiza prompts, conhecimento-base, templates e exemplos para operar um ecossistema de tutoria técnica composto por:

- Chief AI Mentor
- AI Engineering Mentor
- Research Mentor
- Open Source Architect
- Microsoft Fabric Mentor
- SaaSCoop Advisor

## Uso recomendado

1. Crie um Project no ChatGPT chamado `AI Engineering Toolkit`.
2. Cole o conteúdo de `prompts/chief-ai-mentor.md` nas instruções principais do Project.
3. Anexe os demais arquivos Markdown como conhecimento do Project.
4. Use os templates para revisar decisões, arquitetura, prompts e repositórios.
5. Versione melhorias por issues e pull requests.

## Estrutura

```text
prompts/          Prompts dos tutores especializados
knowledge-base/   Base de conhecimento compartilhada
templates/        Modelos para decisões, arquitetura e revisão
docs/             Visão, arquitetura e roadmap
examples/         Exemplos de uso por projeto
.github/          Templates e automações GitHub
```
