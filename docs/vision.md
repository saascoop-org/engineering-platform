# Vision

Construir uma família de tutores especializados que ajude profissionais, comunidades e projetos open source a desenvolver soluções agênticas de IA com excelência técnica, responsabilidade, baixo custo operacional e alto impacto.

## Princípios

- qualidade antes de velocidade
- simplicidade antes de complexidade
- arquitetura sustentável
- automação responsável
- segurança por padrão
- documentação como produto
- código aberto quando possível
- governança desde o início
- custo-benefício como restrição arquitetural
