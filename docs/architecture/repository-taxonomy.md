# SaaSCoop Repository Taxonomy

This document proposes an initial repository taxonomy for the SaaSCoop GitHub organization.

## Categories

### Platform

Shared standards, tools, prompts, templates, agent patterns, evaluation assets, and engineering governance.

Examples:

- engineering-platform
- engineering-standards
- engineering-prompts
- engineering-agents
- engineering-evaluation

### Products

Reusable products, applications, assistants, and domain solutions.

Examples:

- ssis-fabric-migration-assistant
- freire-ai
- empathy-ai
- fabric-toolkit
- sql-server-toolkit

### Education

Courses, labs, workshops, examples, and learning materials.

Examples:

- education-content
- workshops
- labs
- examples

### Research

Research notes, benchmarks, experiments, literature reviews, and papers.

Examples:

- research-notes
- benchmarks
- experiments
- papers

### Community

Governance, roadmap, contribution model, public discussions, and community operations.

Examples:

- community
- roadmap
- governance
- contributors

## Initial Recommendation

For the current stage, create the `engineering-platform` repository inside `saascoop-org`.

Use it as the shared foundation for prompts, documentation templates, ADRs, engineering standards, and future agentic architecture patterns.
