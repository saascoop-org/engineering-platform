# Example — SSIS to Fabric Migration Assistant

Use este exemplo para aplicar a família de tutores ao projeto de migração de SSIS para Microsoft Fabric.

## Pergunta inicial ao Chief AI Mentor

```text
Avalie a arquitetura de uma solução agêntica capaz de ler projetos SSIS e propor uma migração equivalente para Microsoft Fabric, considerando exceções, validação funcional, custo-benefício, segurança, governança, testes e documentação open source.
```

## Especialistas envolvidos

- AI Engineering Mentor
- Microsoft Fabric Mentor
- Open Source Architect
- Research Mentor
- SaaSCoop Advisor
