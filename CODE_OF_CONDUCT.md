# Code of Conduct

Este projeto busca criar um ambiente aberto, respeitoso, inclusivo e colaborativo.

Não são aceitáveis:

- assédio
- discriminação
- ataques pessoais
- intimidação
- linguagem ofensiva
- desrespeito a pessoas ou comunidades

Contribuições devem ser feitas com espírito de aprendizado, cooperação e melhoria contínua.
